# PH413 Comutational Physics 2025 Spring

## Setup

### invironment for jax

- Linux
```bash
    python -m venv ph413-jax
    source ph431-jax/bin/activate
    pip install -r requirements-jax.txt
```

- Windows
```bash
    python -m venv ph413-jax
    source ph431-jax/Scripts/Activate.ps1
    pip install -r requirements-jax.txt
```
